## Run Locally

Clone the project

```bash
  git clone https://github.com/YashSen27/User-Github-Public-Repository-Searcher.git
```

Go to the project directory

```bash
  cd gh-users
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm dev
```
